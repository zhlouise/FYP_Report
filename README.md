**How to cite in LaTeX!**
> 1. You can get the bibtex entry on Google Scholar, or use https://www.doi2bib.org/ to convert doi link to bibtex
> 2. Paste the bibtex entry to the References.bib file in this repo folder
> 3. In Interim_Report.tex file, use \cite{_the first thing after the "{" after your bibtex entry, usually first author last name + year_} wherever you need intext citation
> 4. The reference list will be automatically generated
